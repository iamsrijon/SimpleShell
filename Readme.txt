A simple Shell
~~~~~~~~~~~~~~~~~~~

How to compile and run:
------------------------
1. Download Project.c file and compile using gcc and run the executable
    $ gcc Project.c -o simple_shell
    $ ./simple_shell <custom prompt text>

Then you will see a prompt like Simple_shell% or any custom text provided where you can enter the command you want to execute

To terminate the shell, type "exit" or "quit" and press enter.
